# JavaScript Fundamentals

A collection of JavaScript examples and exercises to help you understand the fundamentals of JavaScript programming.

## Features

- Interactive examples of JavaScript arrays
- Real-time array manipulation
- Random fruit generation using Faker.js
- Modern UI with Tailwind CSS and DaisyUI

## Prerequisites

Before you begin, ensure you have the following installed:

- Node.js (v14 or higher)
- npm (comes with Node.js)
